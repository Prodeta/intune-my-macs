This project is **sample / proof-of-concept code**. It is provided **as-is**, without
warranty of any kind, express or implied.

- **No SLA.** There is no service-level agreement and no support commitment.
- **No production support.** Microsoft does not provide commercial support for this
  repository or its artifacts. Do not open Microsoft Support tickets for issues
  encountered while using this code.
- **No security response process** beyond what is described in [SECURITY.md](SECURITY.md).
- **Best-effort maintenance.** GitHub Issues and Pull Requests are reviewed on a
  best-effort basis by the Microsoft Intune Customer Experience Engineering team. There
  is no guaranteed response time and no guarantee that any issue will be fixed or any
  PR will be merged.
- **Use at your own risk.** Review, test, and adapt every artifact in a non-production
  tenant before considering it for production-managed devices. The maintainers accept
  no responsibility for any impact resulting from use of this code.

## How to get help

- **Questions and bugs:** open a [GitHub Issue](https://github.com/microsoft/intune-my-macs/issues).
- **Security vulnerabilities:** follow [SECURITY.md](SECURITY.md). Do **not** open a public
  issue for security reports.
